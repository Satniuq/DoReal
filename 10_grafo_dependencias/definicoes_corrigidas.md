D_REAL
Real é o campo relacional contínuo de poder-ser no qual as determinações se atualizam segundo limites estruturais, sem admitir exterioridade.

----------

D_SER

Ser é determinação localizada do real portadora de poder-ser cuja atualização manifesta essa determinação.

----------

D_PODER_SER (nível 1 — estrutural absoluto)

Não é “possibilidade”.

É:

dimensão estrutural do real que torna admissível qualquer atualização.

Ou mais seco:

condição estrutural da atualizabilidade do real.

Ele é anterior à distribuição.

----------

D_POTENCIALIDADE (nível 1 — distribuição estrutural)

Não é o poder-ser em si.

É:

conjunto de atualizações estruturalmente admissíveis para uma determinação dada.

Ou seja:

o poder-ser é estrutural

a potencialidade é localizada

----------

D_CAMPO_DE_POTENCIALIDADES (nível 5 — ético)

Já está bem posicionado:

horizonte narrativo de futuros possíveis

dependente de poder-ser + continuidade + ser humano

Este está correto no grafo.

----------

D_ATUALIZACAO

Atualização é a efetivação relacional de uma potencialidade na continuidade do real.

----------  

D_DETERMINACAO

Determinação é a estabilização relacional de uma distinção sob limite estrutural.

----------

D_RELACAO

Relação é a condição estrutural pela qual determinações coexistem e se constituem no real.

----------

D_CONTINUIDADE

Continuidade é a permanência relacional que permite que o ser permaneça sendo.

NOTA: Nos dados, D_RELACAO quase não aparece como dependência formal de D_CONTINUIDADE.

Mas estruturalmente devia aparecer.

Se relação impõe continuidade, então:

D_CONTINUIDADE deveria depender explicitamente de D_RELACAO.

----------

D_CONTINUO

Contínuo é a forma pela qual a continuidade se manifesta como sucessão sem rutura.

NOTA: redundante talvez

----------

D_CAMPO

Campo é a configuração relacional estruturada que torna localizáveis determinações e atualizações no real.

----------

D_LIMITE

Limite é a condição estrutural que diferencia o admissível do ontologicamente impossível.

D_CONTINGENCIA

Contingência é a não-necessidade de uma atualização dentro dos limites do real.

----------

D_NECESSIDADE

Necessidade é o que decorre inevitavelmente da estrutura do real.

----------

D_ESCALA_ONTOLOGICA

Escala ontológica é a diferença de extensão e integração estrutural entre níveis de determinação.

----------

D_IMPOSSIBILIDADE_ONTOLOGICA

Impossibilidade ontológica é a não-admissibilidade estrutural no poder-ser do real.

D_ESTRUTURA

Estrutura é a articulação mínima de distinções que impede a indiferença absoluta.

D_ RELACIONALIDADE_MINIMA

Relacionalidade mínima é a impossibilidade de haver determinação sem relação.

D_NAO_SER

Não-ser é a exclusão determinada implicada por toda a determinação no real.

----------

D_REGULARIDADE

Regularidade é a recorrência estrutural das atualizações no contínuo do real.

----------

D_DISTINCAO

Distinção é a condição mínima de diferenciação sem a qual nada pode ser determinado.
