🧠 O QUE ESTAVA MAL (GENERALIZÁVEL)

O erro não é só do CF08.

É estrutural.

Tipo de erro identificado:

👉 compressão inadequada do problema

Ou seja:

os fragmentos têm densidade
o dossier simplifica mal
a pergunta fica deslocada
🎯 CRITÉRIO GERAL (NOVO)

Este é o critério mais importante de toda a fase:

um dossier só é válido se a sua formulação permitir reconhecer fielmente o padrão descritivo dos fragmentos

Forma negativa (importante)

Um dossier está errado quando:

força os fragmentos a caber nele
usa categorias que não aparecem nos fragmentos
substitui linguagem descritiva rica por abstrações pobres
formula perguntas que o próprio material não sustenta
Forma positiva

Um dossier está correto quando:

a sua pergunta emerge naturalmente dos fragmentos
o vocabulário coincide com o dos fragmentos
as distinções já estavam lá
o problema é reconhecível sem esforço
